# Felidae Ansible role

## What is this?

This repository holds an Ansible role designed to manage the Felidae
ABCI app for a CometBFT cluster to use as its proxyapp.

To set up a CometBFT cluster capable of running Felidae as a proxyapp,
please see our complimentary roles [ansible-role-cometbft-bootstrap](https://github.com/freedomofpress/ansible-role-cometbft-bootstrap) and [ansible-role-cometbft](https://github.com/freedomofpress/ansible-role-cometbft).
